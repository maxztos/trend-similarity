# trend-similarity

## 柱形分布图的相似度分析

### 项目结构说明

show.py：可视化相关

preprocess.py：直方图转换成序列

metrics.py：相似度指标定义（余弦相似度、Pearson指数、DTW距离）

evaluator.py：相似度指标加权计算

程序入口：main.py

输入：

```
main_img: 主图路径(一张)
sub_imgs: 对比图路径列表(支持多张)
```

输出：

```
每张图与主图的相似度得分，以及主图与相似度最高的图提取的数字序列可视化对比
| 分数     | 含义         |
| ------ | ---------- |
| 90～100 | 极其相似（几乎同形） |
| 80～90  | 非常相似       |
| 70～80  | 相似         |
| 60～70  | 勉强相似       |
| < 60   | 差异明显       |

测试用例样本2的得分：
fig1：62
fig2：16
与文档判断基本相同
```
